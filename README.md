# 🛒 Retail Superstore Sales Analysis Dashboard

An end-to-end Data Analytics project built using **Python, PostgreSQL, and Power BI** to analyze sales performance, profitability, customer behavior, and regional trends using the Retail Superstore dataset.

# 🚀 Project Overview

This project demonstrates the complete data analytics workflow—from raw data exploration to SQL analysis and interactive Power BI dashboard development.

The objective was to extract meaningful business insights that help understand sales performance, customer behavior, profitability, and regional performance.

---

# 💡 Key Insights

- Technology generated the highest sales.
- West region recorded the highest revenue.
- California contributed the highest state-wise sales.
- Consumer segment generated the highest profit.
- Higher discounts generally reduced profitability.
